**Sass Fairy** is a set of modules that abstract common functionality to aid in the rapid development of domain specific stylesheets and modules.

There are two groups of modules: Core Modules and Special Modules. **Core Modules** extend the native, [built-in modules](https://sass-lang.com/documentation/modules) provided by Sass. While on the other hand, **Special Modules** are abstractions of other concepts and utilities commonly used when designing stylesheets, mixins, or functions.
