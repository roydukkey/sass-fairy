Provides Sass mixins, functions, and variables for working with breakpoints and aids in responsive development.

## Install

### Requires

* Dart Sass: `>=1.33.0`

Install the package:

```bash
npm install @sass-fairy/break
```

Use the package like any other Sass module:

```scss
@use '@sass-fairy/break';
```
